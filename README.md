This is a hands-on technical demonstration and learning tool built in Go focused on mastering real-time communication using WebSockets.

## Overview

This repository contains a small WebSocket API that manages communication between different machines. It was built using **Go 1.25**.

This is an area of ​​study aimed at testing different ways to create an chat with a main server.

## Technologies used

- Go 1.25
- gorilla/schema for query decoding
- godotenv for environment variables

## Requirements

- Go 1.25 or later installed on your machine
